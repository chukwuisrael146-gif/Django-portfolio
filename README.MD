# 🚀 Israel Chukwu Portfolio

A modern, responsive, and database-driven personal portfolio built with **Django**, **Tailwind CSS**, and **JavaScript**. This project showcases my skills, projects, experience, and technical journey while demonstrating clean architecture, reusable Django applications, and modern UI/UX design principles.

---

## 📖 Overview

This portfolio is more than a personal website—it's a full-stack Django application designed to demonstrate professional software engineering practices. Every major section of the portfolio is organized as a dedicated Django app, making the project scalable, maintainable, and easy to extend.

The design is inspired by premium portfolio experiences while incorporating custom interactions, animations, and reusable components.

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio.git
```

Navigate into the project:

```bash
cd portfolio
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment.

### Windows

```bash
venv\Scripts\activate
```

### macOS / Linux

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Apply migrations:

```bash
python manage.py migrate
```

Run the development server:

```bash
python manage.py runserver
```

---

## 🔧 Environment Variables

Create a `.env` file in the project root.

Example:

```env
SECRET_KEY=your_secret_key

DEBUG=True

DB_NAME=portfolio_db

DB_USER=postgres

DB_PASSWORD=your_password

DB_HOST=localhost

DB_PORT=5432
```
